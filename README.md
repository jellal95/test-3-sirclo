## Setup

- Install Docker
- Copy .env.example to .enc
- Run command `docker-compose up -d`
- Run command `docker-compose exec app composer install`
- Run command `docker-compose exec app php artisan:key generate`
- Run command `docker-compose exec app php artisan migrate:fresh --seed`
- Open http://localhost:82/
